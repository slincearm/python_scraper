# Goodinfo 多項目自動區間爬蟲工具 (Automating Goodinfo Stock Scraper)

這是一個基於 Python 與 Selenium 構建的自動化網頁爬蟲應用程式，專門用於從 Goodinfo 網站持續、穩定地獲取特定股票的歷史區間資料。

## 🌟 功能特色
* **圖形化使用者介面 (GUI):** 使用 `tkinter` 提供簡單易用的圖形介面，使用者可以輕鬆輸入股號、選擇年份上限及爬取項目。
* **多維度資料抓取:** 支援自動獲取「股價 (Price)」、「本淨比 (PBR)」及「本益比 (PER)」三種長期的財務數據。
* **自動環境建置:** 程式啟動時會自我診斷，自動安裝缺失的第三方依賴套件 (如 `selenium`, `webdriver-manager`, `beautifulsoup4`)。
* **無縫接軌連續爬取:** 結合 CDP 攔截檔案路徑，並透過自動化推算日期區間來翻頁，直到達到限制年數或是資料耗盡為止。
* **資料清洗與格式正規化:** 內建強大的二維陣列表格演算法，將混亂的 HTML/XLS 轉換成結構純淨標準的 CSV 檔，完美修復原表因 colspan/rowspan 所造成的錯位現象。
* **非同步背景執行:** 爬蟲邏輯與 I/O 寫入於背景執行緒中運行，確保 GUI 持續保持響應。
* **防封鎖機制:** 特製爬蟲節奏間隔與偽裝選項，降低遭網站封鎖的風險。

## 🚀 快速開始

### 前置要求
* 系統中需具備 **Python 3.x** 環境。
* 使用者電腦需安裝有 **Google Chrome** 瀏覽器。

### 執行方式
無需手動預先安裝套件，直接透過 Python 啟動主程式即可（程式內會自動偵測並 `pip install` 基本需求）：
```bash
python scraper_app.py
```

### 介面操作步驟
1. **輸入股號:** 在「股號」欄位輸入台股股票代碼 (例如：`2330`)。
2. **抓取年數上限:** 設定需要往前追溯的年份數量 (預設為 `5` 年)。
3. **選擇擷取項目:** 勾選需要抓取的資料項目。
4. 點擊綠色的 **「開始執行自動推迴圈爬蟲」** 按鈕。
5. 在下方日誌 (Log) 區塊觀察進度，完成後可至資料夾檢視成果。

## 📂 檔案輸出存放
抓取完成的 CSV 檔案會自動創建於同層目錄的 `Download_Data` 資料夾內，並針對不同數據建立子資料夾儲存：
```text
Download_Data/
├── Price/        # 存放股價歷史資料的 CSV
├── PBR/          # 存放本淨比歷史資料的 CSV
└── PER/          # 存放本益比歷史資料的 CSV
```
CSV 檔案名稱會附帶該檔案抓取的歷史日期區間供快速辨識。

## ⚠️ 已知問題 (Known Issues)
* **日期格式未統一:** 目前從 Goodinfo 下載的「本益比 (PER)」與「本淨比 (PBR)」歷史資料中，交易日期欄位僅顯示「X月Y日」缺少西元年份。雖然嘗試在轉換成 CSV 階段自動反推補齊年份並格式化為與股價資料一致的 `'yy/mm/dd`，但該日期處理邏輯目前尚未成功生效，因此生出的 CSV 中的日期可能仍保持原始網頁的「X月Y日」格式，此問題留待後續版本修復。
